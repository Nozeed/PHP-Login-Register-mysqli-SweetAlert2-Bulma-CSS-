# PHP-Login-Register-mysqli-SweetAlert2-Bulma-CSS-
ระบบ Login และ Register พัฒนาด้วย PHP MySQLi แยกไฟล์ Config/Connect ปลอดภัยด้วยการเช็คข้อมูลซ้ำ แจ้งเตือนสวยงามด้วย SweetAlert2 และดีไซน์ Modern ด้วย Bulma

โครงสร้างไฟล์ (File Structure)
* config.php – เก็บค่าคงที่ DB
* connect.php – ตัวแปรเชื่อมต่อและฟังก์ชันต่างๆ
* index.php – หน้า Login
* register.php – หน้าสมัครสมาชิก
* dashboard.php – หน้าแสดงข้อมูลสมาชิก
* logout.php – ออกจากระบบ
